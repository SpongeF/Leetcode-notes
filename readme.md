# 常用操作

### 提交代码

1. git pull
2. git status
3. git add .
   1. git diff A.py
   2. git add A.py
   3. git diff B.py
   4. git add B.py
      ...
4. git commit -m '注释'
5. git push

### 切换分支

1. git checkout -b wh/branch_a
2. 进行文件修改
3. git add .
4. git commit -m '注释'
5. git push
6. git checkout master (切换回主分支)
